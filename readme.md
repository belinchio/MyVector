# Vector
Реализован небольшой аналог шаблонного класса vector из стандартной библиотеки.  
Данные хранятся в памяти, динамически выделяемой в куче. Память выделяется неинициализированной. Инициализация присходит при фактическом добавении элемента. При добавлении новых элементов, если памяти недостатчно, выделяется новый участок памяти в два раза больше предыдущего, в который перемещаются (или копируются) элементы из старого участка памяти, после чего старый участок памяти удаляется.

## Функционал класса
### Создание вектора
- Конструктор по умолчанию, создаещий пустой вектор:  
`size = 0u`  
`capacity = 0u`  
`buffer = nullptr`  
```cpp
Vector<int> vec;
std::cout << vec.capacity() << " " << vec.size() << std::endl;
```
